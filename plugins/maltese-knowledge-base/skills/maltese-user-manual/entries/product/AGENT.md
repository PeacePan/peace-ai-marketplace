# 一般商品入口 Agent 指引

## 入口資訊
網址：`/checkout`
頁面標題：「一般銷售 POS 平台」

## 介面佈局

一般商品入口使用三欄式佈局：

1. **頂部資訊列：** 左側為會員顯示區；右側為目前操作員資訊與閒置倒數計時（300 秒後自動登出）。

2. **中間欄（商品區）：** 主要操作區，包含：
   - 頂部搜尋列：掃描條碼或輸入商品貨號（提示文字：「點擊輸入商品貨號、國際條碼或直接刷商品」）
   - 購物車 / 商品清單（搜尋列下方）
   - 未開帳時的遮罩提示：「請先至管理面板進行開帳」（僅 CASH_REGISTER 模式）
   - 多項符合搜尋結果時，出現商品選擇器對話框

3. **右側操作面板（Tab 欄）：** 垂直三個 Tab：
   - **結帳**：銷售流程的所有功能按鈕，底部有暫結與小計按鈕
   - **商品**：最多 32 個快速加入按鈕（8 欄 × 4 列），依門市設定配置
   - **管理**：開帳、清帳、報表、列印、支出等管理功能

## 重要限制

### 機台模式
- **CASH_REGISTER（一般收銀）：** 完整功能。銷售前須先開帳。
- **PRICE_ONLY（僅查價）：** 唯讀模式。無法銷售、暫結或結帳。可掃描商品查詢價格，無需開帳。
- **SALON_WORKBENCH（美容工作臺）：** 無法使用商品結帳或暫結（顯示「美容工作機無法使用該功能」）。

### 開/清帳狀態
- 必須先完成開帳（開帳），才能進行任何銷售。
- 未開帳時，中間欄顯示遮罩：「請先至管理面板進行開帳」。
- 需要重新開帳（REOPEN 狀態）時，系統要求先清帳：「請先清帳後再開帳」。
- 跨日未清帳則無法結帳：「跨日作業請進行清帳再進行結帳」。

### 需要設定會員的功能
以下功能須先設定會員：
- 贈品：「非會員無法使用贈品」
- 使用折扣碼：「非會員無法使用折扣碼」
- 整單折扣：「請先設定會員」（外送促銷實驗除外）
- 口袋促銷：「請先設定會員」（外送促銷實驗除外）
- 點數查詢/使用：「請先設定會員」
- 使用優惠券：「請先設定會員」

### 外送訂單限制
管道為 UberEats、FoodPanda 或 Foodomo 時，以下功能停用：
- 使用折扣碼、贈品、加購、點數折抵、點加金活動、使用優惠券

管道為寵速配（FAST_BUY）時，額外停用：
- 整單折扣、口袋促銷

### 優惠券（百分比折扣）互斥規則
已套用百分比折扣型優惠券時，以下功能鎖定：
- 整單折扣開關、使用折扣碼、口袋促銷開關、加購、贈品、點加金活動

須先移除優惠券，才能使用上述功能。

### 換購銷售限制
換購銷售進行中，幾乎所有折扣與促銷功能均停用（含優惠券、整單折扣、折扣碼、口袋促銷、加購、贈品、點數折抵、點加金活動）。換購期間亦無法更換會員。

### 店內專櫃商品
分類為「店內專櫃」的商品不適用大多數促銷與折扣。

### 員工購買
當前會員為員工（員購通路）時，無法使用折扣碼、整單折扣，亦無法購買店內專櫃商品。

### 小計按鈕的行為
點擊「小計」後，系統執行多步驟驗證，接著：
1. 開啟確認對話框（依情境顯示一般確認、換購確認或嚴格確認）
2. 在資料庫建立銷售紀錄
3. 開啟右側付款側邊欄
4. 付款側邊欄顯示商品明細、付款按鈕、未付/已付/溢收金額
5. 全額付清後，點擊「完成結帳」完成交易

### 閒置逾時
操作員在一般商品介面有 300 秒（5 分鐘）的閒置逾時。開啟日業績報表（業績目標）對話框時，計時器暫停。

### 資料自動刷新間隔
- 促銷活動、贈品、整單折扣：每 30 分鐘
- 可列印優惠券與優惠券活動：每 30 分鐘
- 商場促銷設定：每 60 分鐘
- 商場商品設定：每 60 分鐘
- 寵速配訂單通知：每 2 分鐘（僅在 CASH_REGISTER 模式且已開帳時）
