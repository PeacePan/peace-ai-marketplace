# 清帳 Close Register

## 適用情境
每日營業結束後，操作人員須執行清帳作業，確認當日現金與各項交易金額，完成日結流程。

## 操作步驟

### 步驟 1：進入管理面板
在右側操作區點擊「**管理**」頁籤。

### 步驟 2：點擊「清帳」
在管理面板中找到「**清帳**」按鈕並點擊。

- 若目前尚未開帳（已經是清帳狀態），系統會顯示「請先進行開帳」的警告，按鈕不會開啟清帳視窗。
- 若已開帳，系統會開啟清帳對話框。

### 步驟 3：清帳資料載入
開啟清帳對話框後，系統會自動抓取以下資料：
- 當日所有一般商品銷售單
- 當日所有美容服務銷售單
- 當日所有一般退貨單
- 當日所有美容退貨單

載入期間按鈕會處於停用狀態。

### 步驟 4：確認清帳報表
清帳對話框顯示以下區塊：

**報表摘要：**
- 總計金額
- 開櫃櫃金
- 前日留存
- 應繳現金

**收款摘要：**
依付款方式列出各項收款金額，包含：
- 已收金額（paid）
- 溢收金額（overflow）
- 各類付款方式明細（現金、信用卡、離線信用卡等）

**退款摘要：**
- 退款金額
- 折讓金額
- 電子發票相關
- 各類退款方式明細

**支出摘要：**
- 當日支出金額明細

**備註欄：**
可輸入清帳備註文字。

### 步驟 5：確認清帳
確認所有數據無誤後，點擊「**確定**」按鈕完成清帳。

清帳完成後：
- 當日的日結單狀態更新為「已清帳」
- 商品搜尋區域會重新顯示「請先至管理面板進行開帳」的遮罩
- 無法再進行銷售操作，直到下次開帳

## 系統強制清帳提示

若操作人員在需要重新開帳的狀態（REOPEN）下嘗試操作，系統會顯示「請先清帳後再開帳」並自動開啟清帳對話框。

## 跨日未清帳

若開帳後未清帳就跨日繼續操作：
- 點擊小計時會顯示「跨日作業請進行清帳再進行結帳」
- 掃描商品時會顯示「目前為非法開帳日期，請關帳後重新開帳」
- 系統會判斷目前時間是否在機台日結單開帳日的工作日範圍內

## 注意事項 / 限制
- 清帳是不可逆的操作，完成後當日的帳務即告結算。
- 清帳前請確認所有未完成的銷售已處理（包含暫結銷售）。
- 清帳報表中的數據來自所有機台在該門市的銷售紀錄，不僅限於目前操作的機台。
- 清帳後若需回到主選單，可點擊管理面板中的「回主選單」按鈕。
- 大潤發門市（131）的清帳畫面可能有特殊顯示設定。
