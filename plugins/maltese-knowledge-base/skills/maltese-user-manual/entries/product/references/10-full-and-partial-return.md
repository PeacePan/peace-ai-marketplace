# 全額退款與部分退貨 Full and Partial Return

## 適用情境
當客人需要退回已購買的商品並取回款項時，操作人員使用退貨功能。系統支援全部退貨（退回整筆銷售單的所有商品）與部分退貨。

## 全部退貨

### 步驟 1：點擊「全部退貨」
在結帳面板中找到「**全部退貨**」按鈕並點擊。系統會先清除目前的銷售單設定，然後開啟「全部退貨」對話框。

### 步驟 2：搜尋原始銷售單
在全部退貨對話框中，搜尋客人的原始銷售單。可透過銷售單編號、日期等條件查詢。

### 步驟 3：選擇銷售單
從搜尋結果中選擇要退貨的銷售單。系統會顯示該銷售單的完整明細，包含所有商品、數量與金額。

### 步驟 4：確認退貨
確認退貨內容無誤後，系統會進行全額退款處理：
- 退回原始銷售單中所有商品的款項
- 依據原始付款方式進行退款
- 更新原始銷售單狀態為「全額退款」

## 部分退貨

### 步驟 1：點擊「全額退款」按鈕
在結帳面板中找到「**全額退款**」按鈕並點擊。系統會開啟銷售單搜尋對話框。

### 步驟 2：搜尋並選擇銷售單
在搜尋結果對話框中，依銷售單編號、日期等條件找到原始銷售單並選擇。

### 步驟 3：選擇「部分退貨」
在搜尋結果對話框中，切換至「**部分退貨**」標籤或選項（而非全額退款）。系統開啟「部分退貨」對話框。

### 步驟 4：選擇退貨品項
從銷售單的商品列表中，勾選要退回的商品及數量。

### 步驟 4：確認部分退貨
確認選擇的退貨品項後，系統處理部分退款：
- 僅退回選定商品的款項
- 更新原始銷售單狀態為「部分退款」

## 注意事項 / 限制
- 退貨操作需要在已開帳狀態下進行。
- 全部退貨會將原始銷售單的所有商品一次退回，無法選擇性退回部分商品（請使用部分退貨功能）。
- 退款金額會依據原始銷售單的實際支付金額計算，包含已套用的折扣。
- 已退款的銷售單狀態會更新，無法重複退款。
- 退貨完成後，相關的發票會開立折讓單或作廢處理。
