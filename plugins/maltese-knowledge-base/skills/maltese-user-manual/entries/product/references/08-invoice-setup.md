# 發票設定 Invoice Setup

## 適用情境
客人需要開立載有統一編號的發票（公司行號）、使用電子發票載具（手機條碼/自然人憑證）、或需要捐贈發票時，操作人員須在結帳前設定發票方式。

## 操作步驟 — 輸入發票方式

### 步驟 1：點擊「輸入發票方式」
在結帳面板中找到「**輸入發票方式**」按鈕並點擊，開啟「輸入發票方式」對話框。

### 步驟 2：選擇並輸入發票資訊
在對話框中可以設定以下項目：

- **統一編號**：輸入 8 位數的統一編號，適用於公司行號開立二聯式發票
- **電子發票載具條碼**：掃描或輸入客人的手機條碼載具
- **自然人憑證條碼**：掃描或輸入自然人憑證條碼
- **捐贈碼**：輸入捐贈機構的愛心碼

### 步驟 3：確認設定
輸入完成後確認，發票方式會保存在目前的銷售設定中，直到結帳完成或手動清除。

## 操作步驟 — 清除發票方式

若需要取消或修改已設定的發票方式：

點擊「**清除發票方式**」按鈕，系統會立即清除目前設定的所有發票資訊（統一編號、載具條碼、捐贈碼等）。

## 發票列印規則

結帳時系統會依據以下規則決定是否列印發票明細：
- 有統一編號且金額 > 0：預設列印明細
- 無統一編號且金額 > 0：預設不列印明細
- 金額 = 0：一律列印明細

操作人員可在付款視窗中手動切換「列印明細」開關。

## 異業合作統一編號

若客人購買的是異業合作商品（例如特定合作品牌），系統會檢查是否已輸入該合作方指定的統一編號。若未輸入或輸入的統一編號與要求不符，點擊小計時會顯示提示訊息，要求輸入正確的統一編號。

## 注意事項 / 限制
- 發票方式在暫結時會一併保存，取回暫結後仍然有效。
- 清除發票方式的操作無需確認，點擊即生效。
- 統一編號與載具條碼可同時設定，但實際開立發票時會依據財政部規定處理。
