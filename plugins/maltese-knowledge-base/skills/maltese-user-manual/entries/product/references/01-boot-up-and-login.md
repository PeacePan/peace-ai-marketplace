# 開機與登入 Boot-Up and Login

## 適用情境
每日首次啟用 POS 系統，或因瀏覽器重新開啟需要重新登入時，操作人員須完成開機程序才能進入銷售畫面。

## 操作步驟

### 步驟 1：開啟 POS 應用程式
在瀏覽器中開啟 POS 系統首頁。頁面標題顯示「萬達寵物 開機程序」，畫面中央會出現三步驟導引列：
1. **使用者登入**
2. **選擇門市與機台**
3. **選擇操作介面**

系統會先確認您是否已登入。若尚未登入，畫面會顯示讀取圖示後進入登入表單。

### 步驟 2：輸入金鑰登入
登入表單包含兩個欄位：
- **金鑰編號**：輸入您的存取金鑰 (Access Key ID)
- **金鑰密碼**：輸入您的私密金鑰 (Secret Access Key)，此欄位以密碼遮罩顯示

兩個欄位都填寫完成後，點擊「**登入**」按鈕送出。若其中一個欄位為空，登入按鈕會處於停用狀態。

### 步驟 3：選擇門市與機台
登入成功後，進入第二步「選擇門市與機台」畫面，需要填寫三個下拉選單：

1. **選擇門市**：從門市清單中選擇您所在的門市。若無可選擇的門市，下拉選單會顯示「無可選擇門市」並處於停用狀態。
2. **選擇機台**：選定門市後，系統會載入該門市可用的機台列表。在門市尚未選定或機台資料載入中時，此欄位處於停用狀態。
3. **選擇熱感式列表機**：選擇收據列印機型號，可選項為 **Zebra** 或 **Citizen**。

三個欄位都填寫完成後，點擊「**確定**」按鈕繼續。

### 步驟 4：選擇操作介面
最後一步顯示三個大按鈕，對應三種操作介面：

| 按鈕文字 | 對應路徑 | 說明 |
|---------|---------|------|
| 前往 **一般商品結帳** | `/checkout` | 一般商品的銷售結帳平台 |
| 前往 **美容商品結帳** | `/salon` | 美容服務的銷售結帳平台 |
| 前往 **美容工作臺** | `/workbench` | 美容師的工作排程管理 |

點擊其中一個按鈕後，系統標記開機程序完成，隨即跳轉至對應的操作畫面。

## 直接 URL 存取（跳過開機程序）

若操作人員直接在瀏覽器輸入功能頁面的網址（例如直接輸入 `/checkout`），系統會進行以下檢查：

1. **未登入**：顯示讀取中畫面。
2. **已登入但開機程序未完成**（門市/機台尚未選擇）：系統自動跳轉回首頁 (`/`) 重新進行開機程序。
3. **已登入且開機程序已完成**：正常進入該頁面。

## 注意事項 / 限制
- 金鑰編號與金鑰密碼由系統管理員提供，請妥善保管。
- 開機程序完成前無法進行任何銷售操作。
- 若需切換門市或機台，需回到主選單重新執行開機程序。
- 頁面標題會根據環境（正式/測試）顯示不同標記，方便辨識目前所在環境。
