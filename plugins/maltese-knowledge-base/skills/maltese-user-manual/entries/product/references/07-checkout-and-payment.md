# 結帳與付款 Checkout and Payment

## 適用情境
操作人員將所有商品加入購物車並完成優惠設定後，透過「小計」開始結帳流程，選擇付款方式完成交易。

## 操作步驟

### 步驟 1：點擊「小計」按鈕
在結帳面板底部點擊粉色的「**小計**」按鈕。按鈕的左方顯示目前購物車的小計金額、折扣金額與應付總額。

系統會在按下小計時進行以下檢查（任一不通過則無法繼續）：

- 必須已開帳
- 購物車中必須至少有一件商品
- 最佳促銷組合計算必須已完成
- 若有跨日未清帳的情形，須先清帳
- 寵速配訂單的商品總額必須與訂單金額一致
- 點數折抵金額不可超過可折抵上限
- 異業合作商品必須使用對應的專屬會員並獨立結帳
- 員購不可購買店內專櫃料件
- 不得銷售獸醫專用（VET）商品
- 寵速配與外送訂單至少需有一筆購物袋料件
- 購物袋數量不可超過 10 筆
- 客訂商品必須包含在銷售單中
- 分批取兌換核銷必須完成

### 步驟 2：商品數量確認
通過檢查後，系統會顯示確認對話框（依情境不同有不同場景）：

- **一般銷售**：開啟「商品數量確認」畫面，請確認各商品的數量是否正確
- **換購銷售**：開啟「換購銷售確認」畫面
- **外送/寵速配訂單**：開啟「嚴格商品數量確認」畫面
- **特定期間免確認數量**：直接建立銷售單並開啟付款視窗

確認無誤後，系統會建立銷售單並開啟付款側邊欄。

### 步驟 3：選擇付款方式
付款側邊欄從右側滑出，分為上下兩個區域：

**上方 — 購物車與付款明細：**
- 商品列表：顯示所有商品的名稱、數量、單價、小計
- 已付款列表：顯示已選擇的各付款方式與金額
- 找零資訊

**下方 — 付款按鈕與統計：**

付款資訊區顯示三個數字：
- **未付**：尚需支付的金額
- **已付**：已輸入的付款金額
- **溢收**：超額付款的金額

右下角顯示小計、折扣、總額與商品數量。

### 步驟 4：輸入付款金額
可用的付款方式按鈕如下：

**現金付款：**
| 按鈕 | 說明 |
|------|------|
| 現金 | 全額現金付款 |
| 現金 500 | 付款 500 元 |
| 現金 1000 | 付款 1,000 元 |
| 現金 1500 | 付款 1,500 元 |
| 現金 2000 | 付款 2,000 元 |

**信用卡付款：**
| 按鈕 | 說明 |
|------|------|
| 離線信用卡（全額） | 以離線方式刷全額信用卡 |
| 離線信用卡（指定金額） | 輸入指定金額後刷離線信用卡 |
| 信用卡（全額） | 以連線方式刷全額信用卡 |
| 信用卡（指定金額） | 輸入指定金額後刷連線信用卡 |

**第三方付款：**
| 按鈕 | 說明 |
|------|------|
| 宜睿 | 宜睿票券（Edenred）支付 |
| 胖達狗 支付 | PandaGo 支付 |
| One 碼 | 台新 One 碼支付 |

**外送平台離線付款：**
| 按鈕 | 說明 |
|------|------|
| 寵速配 離線（全額） | 寵速配訂單結帳 |
| UberEats 離線（全額） | UberEats 訂單結帳 |
| FoodPanda 離線（全額） | FoodPanda 訂單結帳 |
| Foodomo 離線（全額） | Foodomo 訂單結帳 |

**其他（特定門市）：**
- **大潤發員購**：僅大潤發門市顯示
- **標記為客訂單**：僅啟用客訂功能的門市顯示

### 步驟 5：混合付款（分開付款）
可以組合多種付款方式。例如先用信用卡付一部分，再用現金付剩餘金額。

每次選擇付款方式後，「未付」金額會減少，「已付」金額會增加。當「未付」歸零時即可完成結帳。

**注意事項：**
- 同一筆銷售僅能使用一張信用卡
- One 碼不得搭配其他付款方式
- 宜睿票券不得與 PandaGo 或 One 碼搭配
- 已付款項目若為現金，可以點擊刪除按鈕移除；信用卡、宜睿票券等已核銷的付款無法刪除

### 步驟 6：完成結帳
當金額付清後（未付金額為 0 或現金溢收），點擊「**完成結帳**」按鈕：
- 若有現金付款，系統會自動開啟錢櫃
- 系統完成銷售單更新
- 列印收據（依據列印明細設定）
- 發票開立

### 列印明細按鈕
付款視窗中有「列印明細」開關：
- 金額 > 0 且有統一編號時，預設為**開啟**
- 金額 > 0 且無統一編號時，預設為**關閉**
- 金額 = 0 時，一律為**開啟**

操作人員可手動切換此設定。

### 返回（取消小計）
在付款視窗中點擊「**返回**」按鈕可以取消本次結帳：
- 若尚未有任何付款，直接返回購物車畫面，銷售單會被取消
- 若已有部分付款（現金），需先從銷售單上刪除付款方式
- 若已使用宜睿票券等已核銷的付款，無法返回，需完成結帳後再進行退款

## 注意事項 / 限制
- 查價機模式（PRICE_ONLY）與美容工作機無法使用結帳功能。
- 專櫃門市無法使用：信用卡（線上/離線）、宜睿票券、One 碼等電子支付方式，需至專櫃 POS 機進行。
- 小計按鈕有防連擊機制，點擊後會短暫停用避免重複提交。
- 結帳過程中若發票機未連線，換購銷售或特定期間的銷售將被攔截。
