# 搜尋與加入商品 Search and Add Items

## 適用情境
操作人員需要將客人欲購買的商品加入購物車，可透過條碼掃描、手動輸入貨號或快捷按鈕等方式操作。

## 操作步驟

### 方式一：條碼掃描（主要方式）

#### 步驟 1：確認搜尋列已聚焦
畫面中央上方有一個搜尋列，提示文字為「點擊輸入商品貨號、國際條碼或直接刷商品」。系統會在以下條件下自動聚焦到搜尋列：
- 已有門市人員登入
- 沒有任何對話框或側邊欄開啟中

#### 步驟 2：使用掃碼槍掃描商品條碼
掃碼槍會自動將條碼輸入搜尋列並送出。系統處理邏輯如下：

- **找到唯一商品**：自動加入購物車，搜尋列清空。
- **找到多筆商品**：彈出「商品選擇器」對話框，顯示所有符合的商品，您需要勾選正確的商品後確認。
- **找不到商品**：顯示「根據 [條碼] 找不到商品」的警告訊息。

#### 步驟 3：連續掃描同一商品
若短時間內連續掃描同一條碼，系統會自動累加數量。例如連續掃描同一商品 3 次，該商品會以數量 3 加入購物車。系統使用 250 毫秒的延遲機制來批次處理連續掃描。

### 方式二：手動輸入貨號

#### 步驟 1：點擊搜尋列
手動點擊搜尋列使其聚焦。

#### 步驟 2：輸入商品貨號或國際條碼
在搜尋列中輸入商品的貨號（name）或國際條碼（code），然後按下 Enter 鍵送出。後續流程與條碼掃描相同。

### 方式三：使用「數量 x 商品」功能

在結帳面板中點擊「**數量 x 商品**」按鈕，可以先指定數量再掃描或輸入商品，一次加入指定數量的商品。

### 方式四：使用商品快捷鍵

#### 步驟 1：切換到「商品」頁籤
在右側操作區點擊「**商品**」頁籤。

#### 步驟 2：點擊快捷按鈕
畫面顯示最多 32 個快捷按鈕，排列為 8 列 x 4 行的網格。每個按鈕對應預設好的一組商品（可包含多個品項與指定數量）。按鈕由後台設定，不同門市群組可能顯示不同的快捷按鈕。

點擊按鈕後，系統自動搜尋並加入對應的商品到購物車中。

## 商品加入後的行為

### 促銷提示
當加入的商品符合以下任何活動時，系統會在畫面上方顯示提示訊息：
- **商品促銷**：顯示「此商品符合最新商品促銷」及促銷活動名稱
- **加購活動**：提示有可用的加購優惠
- **贈品活動**：提示有可用的贈品
- **折扣活動**：提示有可用的折扣

注意：若目前設定的身份為**員工**，則不會顯示促銷提示。

### 時價商品
若加入的商品為時價商品（open price），系統會自動彈出「時價設定」視窗，要求您輸入該商品的實際售價。此視窗無法透過點擊背景或按 Escape 關閉，必須完成價格設定。

### 調整購物車中的商品
商品加入購物車後，您可以點擊購物車中的商品項目進行以下操作：
- 調整數量
- 移除商品
- 查看商品詳細資訊

### 分批取兌換
若在搜尋列刷入以「MAPE」開頭的字串，系統會視為分批取兌換編號：
- 驗證成功：顯示「分批取兌換編號已帶入！請將所有商品刷入後完成核銷動作」
- 驗證失敗：顯示不合法原因

## 注意事項 / 限制
- 未開帳狀態下（查價機模式除外），掃描商品會被攔截並顯示警告。
- 當系統正在帶入銷售資料（暫結、換購等）時，掃描商品會顯示「銷售資料帶入中，請稍候再操作」。
- 搜尋列在有對話框開啟時不會自動聚焦，避免干擾對話框操作。
- 快捷按鈕的內容由後台管理，門市人員無法自行修改。
- 購物袋數量不可超過 10 筆。
