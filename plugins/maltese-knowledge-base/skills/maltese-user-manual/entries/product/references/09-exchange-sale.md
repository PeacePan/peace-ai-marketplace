# 換購銷售 Exchange Sale

## 適用情境
當客人需要將已購買的商品進行換購（退回原商品並購買新商品）時，操作人員使用換購銷售功能。

## 操作步驟

### 步驟 1：點擊「換購銷售」
在結帳面板中找到「**換購銷售**」按鈕並點擊。系統會先清除目前的銷售單設定，然後開啟「換購銷售」對話框。

### 步驟 2：搜尋原始銷售單
在換購銷售對話框中，搜尋客人原始的銷售單。可透過銷售單編號、日期等條件進行查詢。

### 步驟 3：選擇原始銷售單
從搜尋結果中選擇要進行換購的原始銷售單。系統會帶入原始銷售單的相關資訊。

### 步驟 4：調整購物車
原始銷售單的資料帶入後：
- 購物車中會顯示原始銷售的商品
- 操作人員可以移除要退回的商品
- 可以加入新的換購商品

### 步驟 5：完成換購結帳
調整完畢後，點擊「小計」按鈕。系統會開啟「換購銷售確認」畫面，確認換購明細後進入付款流程。

結帳時系統會先確認發票機連線狀態，若未連線則無法繼續。

## 換購銷售的限制

換購銷售進行中時，以下功能會被限制：

| 功能 | 限制 |
|------|------|
| 使用優惠券 | 不可使用（「換購不可使用優惠券」） |
| 整單折扣 | 不可使用（「換購銷售無法加入整單折扣」） |
| 使用折扣碼 | 不可使用（「換購銷售無法加入折扣碼」） |
| 口袋促銷 | 不可使用（「換購銷售無法加入口袋促銷」） |
| 加購 | 不可使用（「換購銷售無法加入加購」） |
| 贈品 | 不可使用（「換購銷售無法加入贈品」） |
| 點數折抵 | 不可使用（「換購不可使用點數」） |
| 點加金活動 | 不可使用（「換購不可使用點加金活動」） |
| 更換會員 | 不可更換（「換購時無法進行更換會員」） |

## 換購銷售的暫結

若換購銷售需要暫時中斷，可以使用暫結功能：
- 暫結時會保存原始銷售單編號與促銷快照
- 取回暫結後，換購銷售的關聯狀態會完整恢復

## 注意事項 / 限制
- 換購銷售的小計檢查比一般銷售更嚴格，發票機必須處於連線狀態。
- 換購銷售結帳後，原始銷售單的狀態會相應更新。
- 換購銷售不會觸發二次驗證（促銷/贈品/整單折扣的限制使用組數檢查），因為大部分優惠功能已被禁用。
