# 設定會員 Set Member

## 適用情境
當客人為萬達寵物會員時，操作人員需要在結帳前設定會員身份，以便使用會員相關的優惠功能與累積消費紀錄。

## 操作步驟

### 步驟 1：開啟會員查詢
在畫面上方的會員區域，若目前無會員設定，會顯示一個遮罩覆蓋的「**查詢會員**」按鈕。點擊此按鈕（或點擊遮罩的任意位置）即可開啟「會員查詢」對話框。

### 步驟 2：輸入查詢條件
在會員查詢對話框中，可以透過以下方式搜尋會員：
- 會員編號
- 電話號碼
- 會員姓名

輸入完畢後送出查詢。

### 步驟 3：從搜尋結果中選取會員
若搜尋結果有多筆符合的會員，從列表中選取正確的會員。選取後會員資料會顯示在畫面上方的會員區域。

## 會員資料顯示

設定會員後，畫面上方會顯示以下資訊：

**左半部：**
- **會員編號**：會員的系統編號
- **會員名稱**：會員姓名。若為員工身份，名稱旁會顯示「員工」標籤；若為一般會員，會顯示會員等級標籤
- **上一次消費時間/門市（一般）**：最近一次在一般商品結帳的消費日期與門市
- **上一次消費時間/門市（美容）**：最近一次在美容結帳的消費日期與門市

**右半部（僅一般會員顯示）：**
- **點數**：目前可用點數
- **門市券**：可用的門市優惠券張數
- **美容券**：可用的美容優惠券張數
- **升級/維持差額**：依據會員等級顯示不同資訊
  - 經典卡：距離升級金卡差額
  - 金卡：距離升級黑卡差額、維持金卡差額
  - 黑卡：維持黑卡差額

**功能按鈕（僅已設定會員時顯示）：**
- 「**會員、寵物資料**」：查看會員詳細資料、寵物資訊及點數紀錄
- 「**歷史消費**」：查看會員的歷史消費紀錄

## 更換會員

點擊會員資料的左半部區域可以更換會員：

- **購物車為空**：直接開啟會員查詢對話框。
- **購物車有商品或已使用優惠券/點數折抵**：系統會先彈出確認視窗，詢問是否要將目前的購物車內容加入暫結後再更換會員。
- **換購銷售中**：無法更換會員，顯示「換購時無法進行更換會員」。
- **美容結帳中**：無法更換會員，顯示「美容暫不開放更改會員的功能」。
- **異業合作會員**：若目前會員為異業合作專屬會員，且購物車中有該異業合作的商品，則無法更換。

## 需要設定會員才能使用的功能

以下功能必須先設定會員才能操作，否則會顯示「請先設定會員」或「非會員無法使用」：

| 功能 | 錯誤提示 |
|------|---------|
| 使用折扣碼 | 「非會員無法使用折扣碼」 |
| 贈品 | 「非會員無法使用贈品」 |
| 整單折扣 | 「請先設定會員」 |
| 口袋促銷 | 「請先設定會員」 |
| 點數查詢/使用 | 「請先設定會員」 |
| 使用優惠券 | 「請先設定會員」 |

注意：若門市已啟用外送平台促銷實驗功能，整單折扣與口袋促銷在外送訂單情境下可不需設定會員。

## 注意事項 / 限制
- 員工身份的會員在結帳時會走員購通路，部分功能（如折扣碼、整單折扣）不適用。
- 異業合作會員只能購買對應的異業合作商品，無法購買其他商品。
- 會員等級分為：經典卡（CLASSIC）、金卡（GOLD）、黑卡（BLACK）。金卡與黑卡名稱旁會顯示星號標記。
