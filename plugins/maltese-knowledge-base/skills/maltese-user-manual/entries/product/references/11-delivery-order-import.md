# 外送訂單帶入 Delivery Order Import

## 適用情境
當門市收到外送平台（UberEats、FoodPanda、Foodomo）或寵速配的訂單時，操作人員需要將訂單資料帶入 POS 系統進行結帳。

## 操作步驟

### 步驟 1：點擊「外送訂單帶入」
在結帳面板中找到「**外送訂單帶入**」按鈕（雙倍高度，灰色）並點擊。系統開啟外送訂單帶入對話框。

### 步驟 2：選擇外送平台
對話框標題為「外送訂單帶入｜選擇外送平台」，顯示以下平台選項：

| 平台 | 圖示 |
|------|------|
| **UberEats** | UberEats 商標圖示 |
| **FoodPanda** | FoodPanda 商標圖示 |
| **Foodomo** | Foodomo 商標圖示 |
| **寵速配** | 寵速配商標圖示 |

點擊對應的平台圖示進入該平台的訂單匯入流程。

### 步驟 3：匯入訂單資料
根據選擇的平台，進入各平台專屬的訂單匯入介面：

- **UberEats**：輸入或掃描 UberEats 訂單資訊
- **FoodPanda**：輸入或掃描 FoodPanda 訂單資訊
- **Foodomo**：輸入或掃描 Foodomo 訂單資訊
- **寵速配**：從寵速配系統帶入待處理的訂單

### 步驟 4：確認訂單帶入
訂單資料匯入後，系統會將訂單中的商品自動加入購物車，並設定對應的通路別。

## 訂單帶入後的限制

外送訂單帶入後，以下功能會受到限制：

### 通用限制（所有外送訂單）
| 功能 | 限制 |
|------|------|
| 使用折扣碼 | 「通路別為 UberEats、FoodPanda、Foodomo 時，無法使用此功能」 |
| 加購 | 「通路別為 UberEats、FoodPanda、Foodomo 時，無法使用此功能」 |
| 贈品 | 「通路別為 UberEats、FoodPanda、Foodomo 時，無法使用此功能」 |
| 點數折抵 | 「通路別為 UberEats、FoodPanda、Foodomo 時，無法使用此功能」 |
| 點加金活動 | 不可使用 |
| 使用優惠券 | 不可使用 |

### 寵速配專屬限制
| 功能 | 限制 |
|------|------|
| 整單折扣 | 「因寵速配訂單需由系統帶入... 無法使用此功能」 |
| 使用折扣碼 | 同上 |
| 口袋促銷 | 同上 |
| 加購 | 同上 |
| 贈品 | 同上 |
| 點數折抵 | 同上 |

### 系統帶入的外送訂單限制
當訂單是由系統自動帶入（非手動輸入）時，口袋促銷功能也會被限制：
- 「系統帶入 UberEats、FoodPanda、Foodomo 訂單時，無法使用此功能」

## 結帳注意事項

### 購物袋要求
寵速配及外送訂單結帳時，購物車中**至少需要有一筆購物袋料件**，否則點擊小計時會顯示警告。

### 寵速配金額核對
寵速配訂單的商品總額必須與訂單上的金額一致，否則無法進行小計。系統會顯示「請確認商品數量、商品總金額與訂單資料是否相同！」

### 付款方式對應
各平台訂單結帳時需使用對應的付款方式：
- UberEats 訂單 → 「UberEats 離線（全額）」
- FoodPanda 訂單 → 「FoodPanda 離線（全額）」
- Foodomo 訂單 → 「Foodomo 離線（全額）」
- 寵速配訂單 → 「寵速配 離線（全額）」

### 嚴格商品數量確認
外送訂單與寵速配訂單在點擊小計後，會進入嚴格的商品數量確認畫面，必須確認每件商品的數量正確無誤後才能進入付款。

## 注意事項 / 限制
- 外送訂單帶入前，系統會檢查開帳狀態，未開帳則無法選擇平台。
- 店內專櫃商品不適用寵速配訂單。
- 專櫃門市不適用外送平台交易。
- 寵速配新訂單到來時，系統會每 2 分鐘檢查一次並播放提示音「寵速配新訂單來囉！請儘速至公務機接單！」。
