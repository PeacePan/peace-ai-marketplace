# 美容入口 Agent 指引

## 入口資訊
- 網址：`/salon`
- 頁面標題：「萬達寵物 美容服務 POS 平台」

## 介面佈局

美容入口採三欄佈局：

| 區域 | 位置 | 內容 |
|------|------|------|
| 資訊列 | 上方 | 左側為會員資訊（含寵物列表、點數、等級）；右側為操作員資訊（含閒置倒數 30 分鐘） |
| 服務項目列表 | 中間 | 依寵物分組顯示已新增的服務項目，美容服務與使用包月服務會顯示排定日期；下方顯示整單折扣列表與點數禮物卡列表；未開帳時顯示遮罩提示 |
| 操作面板 | 右側 | 垂直分頁：「結帳」與「管理」兩個 Tab |

## 與一般商品入口的差異

- **必須先設定會員**：一般商品入口可直接掃碼，美容入口必須先設定會員才能新增服務。
- **銷售對象為服務**：非實體商品，無條碼掃描，需手動選擇服務分類與項目。
- **服務依寵物分組**：每隻寵物獨立列出其服務項目。
- **僅兩個 Tab**：結帳與管理（無「商品」快捷分頁）。
- **排定日期**：美容服務與使用包月服務有排定日期概念。
- **後付款機制**：可建立未付款銷售稍後付款；工作臺模式僅允許後付款。
- **閒置逾時 30 分鐘**：一般商品入口為 5 分鐘。
- **包月服務**：支援購買與使用包月/套餐服務，有專屬的退款流程。
- **促銷系統獨立**：使用美容專屬促銷（每 30 分鐘自動更新），與一般商品促銷分開。

## 重要限制

- 未開帳時無法新增服務（查價機模式除外），中間欄顯示遮罩。
- 未設定會員時無法新增服務、使用折扣碼、使用點數、使用優惠券。
- 工作臺模式（美容工作機）僅能使用後付款，小計僅允許 0 元銷售。
- 查價機模式無法使用小計功能。
- 跨日未清帳時無法結帳，需先進行清帳再結帳。
- 購買包月服務與使用包月服務不可同時存在於購物車。
- 使用未付款包月服務時，購物車總金額必須為 0 元。
- 員工（員購）無法購買包月服務、無法使用折扣碼。
- 每隻寵物的單次洗澡與單次美容各限一組。
- 每隻寵物在同一門市的有效包月服務上限 15 組。
- ONE 碼必須全額支付，不得混搭（宜睿票券除外的情況下不可混搭）。
- 專櫃門市無法使用信用卡、One 碼、宜睿票券。
- 宜睿票券不適用購買包月、安親、住宿。
- 退款期間對話框鎖定，直到退款完成。
- ONE 碼超過 180 天無法退款。
