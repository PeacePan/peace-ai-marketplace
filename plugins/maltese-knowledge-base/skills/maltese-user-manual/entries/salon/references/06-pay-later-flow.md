# 後付款流程

## 適用情境
操作員需要先建立服務銷售紀錄，但暫時不進行付款。常見於：美容師開始服務時先建單，待服務完成後再由櫃台人員收款；或工作臺模式的機台（僅允許後付款，無法直接結帳）。

## 操作步驟

### 步驟 1：新增服務項目
先完成會員設定並新增至少一項服務項目（詳見「新增/修改服務」說明）。

### 步驟 2：點擊「後付款」按鈕
在結帳分頁底部，點擊「後付款」按鈕（藍色，位於左側，佔 35% 寬度）。

### 步驟 3：系統驗證與建立銷售
系統會依序執行：
1. 檢查是否已開帳（未開帳會跳出開帳提示）
2. 檢查是否已新增服務（未新增服務會提示「請先加入服務」）
3. 驗證服務項目的合法性
4. 建立或更新銷售單（狀態為「未付款」）

### 步驟 4：確認完成
成功後，系統顯示「此筆銷售已加入未付款銷售清單中」，畫面自動清空，準備下一筆銷售。

---

## 取回未付款銷售進行付款

未付款銷售可透過兩種方式取回：

### 方式 1：查詢未付款銷售
1. 在結帳分頁中，點擊「查詢未付款銷售」按鈕
2. 從清單中選擇要取回的銷售
3. 點擊「加入銷售」按鈕
4. 系統將該筆銷售的服務項目載入畫面
5. 點擊「小計」按鈕完成付款

### 方式 2：工單列表
1. 在結帳分頁中，點擊「工單列表」按鈕
2. 從今日有效工單中找到目標銷售
3. 查看銷售詳情

---

## 更新未付款銷售

若取回未付款銷售後修改了服務內容：
- 再次點擊「後付款」：系統會更新該筆未付款銷售
- 若未做任何修改，直接點擊「後付款」不會重複建立銷售

---

## 工作臺模式

### 重要限制
工作臺模式的機台（美容工作機）**只能使用後付款**，無法點擊「小計」直接結帳。

若在工作臺模式下點擊「小計」且銷售金額不為 0 元，系統會提示：
> 「美容工作機僅能成立 0 元銷售，請點擊【後付款】」

唯一例外：當銷售總金額為 0 元時，工作臺模式可使用「小計」完成銷售。

---

## 注意事項
- 後付款操作期間，所有按鈕會暫時禁用，避免重複提交。
- 若有多個系統處理程序同時進行（如計算促銷、載入未付款銷售等），後付款按鈕也會暫時禁用。
- 使用包月服務的銷售如果原購買包月服務尚未付款，建議先透過後付款建單，並優先付清原購買包月服務的銷售。
