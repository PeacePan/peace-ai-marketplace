# 美容折扣與優惠

## 適用情境
操作員需要為美容服務銷售套用各種折扣或優惠，包含優惠券、促銷活動、整單折扣、會員點數折抵及折扣碼。

---

## 使用優惠券

### 步驟 1：點擊「使用優惠券」按鈕
在結帳分頁中，點擊「使用優惠券」按鈕。

### 步驟 2：操作優惠券
系統開啟優惠券對話框，依畫面指引操作。

### 注意事項
- 必須先設定會員才能使用優惠券。未設定會員時點擊會提示「請先設定會員」。
- 美容入口使用的是「美容優惠券」，與一般商品入口的優惠券不同。

---

## 美容促銷活動列表

### 步驟 1：點擊「活動列表」按鈕
在結帳分頁中，點擊「活動列表」按鈕（綠色）。

### 步驟 2：查看促銷活動
系統顯示目前有效的美容促銷活動列表，以「促銷活動」分頁呈現。活動排序規則：
- 會員適用的活動排在員工適用的之前
- 同類型依建立時間由新到舊排序

### 注意事項
- 此列表僅供查閱，促銷活動會在結帳時由系統自動計算最佳組合套用。
- 系統每 30 分鐘會自動更新促銷活動資訊。

---

## 使用會員點數折抵

### 步驟 1：點擊「點數查詢/使用」按鈕
在結帳分頁中，點擊「點數查詢/使用」按鈕。

### 步驟 2：查看並使用點數
系統顯示會員目前的可用點數，依畫面指引設定折抵金額。

### 注意事項
- 必須先設定會員才能使用點數。未設定會員時會提示「請先設定會員」。
- 必須先新增至少一項服務。無服務時會提示「無銷售資料，無法使用點數」。
- 若已有會員點數折抵的整單折扣，會提示「已有會員點數折抵之整單折扣，若要異動請先刪除」。

---

## 使用折扣碼

### 步驟 1：點擊「使用折扣碼」按鈕
在結帳分頁中，點擊「使用折扣碼」按鈕。

### 步驟 2：輸入折扣碼
在對話框中輸入折扣碼。

### 注意事項
- 必須先設定會員。非會員點擊時會提示「非會員無法使用折扣」。
- 員購身份無法使用折扣碼，會提示「員購無法使用折扣碼」。

---

## 點數禮物卡

### 步驟 1：點擊「點數禮物卡」按鈕
在結帳分頁中，點擊「點數禮物卡」按鈕（橘色）。此按鈕僅在目前無未付款銷售時顯示。

### 步驟 2：存入禮物卡點數
依畫面指引完成點數禮物卡的存入操作。

### 注意事項
- 當目前有未付款銷售時，此按鈕位置會替換為「取消銷售」按鈕。
- 存入禮物卡點數後，若銷售列表為空且無其他點數紀錄，系統會自動清空會員。

---

## 折扣在中間欄的顯示

套用折扣後，中間欄的服務項目列表下方會顯示：
- 整單折扣列表：顯示已套用的折扣明細
- 點數禮物卡列表：顯示已存入的點數紀錄

---

## 互斥規則摘要

| 情境 | 限制 |
|------|------|
| 未設定會員 | 無法使用優惠券、點數折抵、折扣碼 |
| 員購身份 | 無法使用折扣碼 |
| 無服務項目 | 無法使用點數折抵 |
| 已有點數折抵 | 需先刪除才能重新設定 |
| 有未付款銷售 | 點數禮物卡按鈕替換為取消銷售按鈕 |
