# 全額退款與包月退款

## 適用情境
操作員需要退還已完成的美容服務銷售（全額退款），或退還包月/套餐服務（包月退款）。

---

## 1. 全額退款

### 適用情境
退還一筆已付款的美容服務銷售的全部金額。

### 操作步驟

#### 步驟 1：點擊「全部退貨」按鈕
在結帳分頁中，點擊「全部退貨」按鈕（紫色）。

#### 步驟 2：搜尋銷售
在搜尋畫面中，可透過以下方式搜尋目標銷售：
- 輸入文字搜尋：發票號碼、銷售編號、會員編號、員工編號、手機號碼、康記額外聯絡資訊
- 日期區間搜尋：指定付款日期的起始與結束日期

#### 步驟 3：選擇銷售
系統列出符合條件的銷售（狀態為已付款、部分退款或全部退款），選取要退款的銷售。

#### 步驟 4：二次確認
系統進入二次確認畫面，確認退款資訊無誤後，點擊確認。

#### 步驟 5：驗證發票條碼（視情況）
若銷售有發票號碼且無共通性載具，系統會要求驗證發票條碼。

#### 步驟 6：退款作業
系統建立銷退單。若銷售的帳務類型為收據，退款立即完成。否則進入退款操作畫面：

1. 畫面顯示原始付款方式與金額
2. 逐筆選擇退款方式並輸入退款金額
3. 每新增一筆退款紀錄，系統即時更新
4. 完成所有退款後，點擊完成

#### 步驟 7：折讓單/發票作廢
系統會根據以下規則處理發票：
- **同發票期間內退款**：作廢發票
- **跨發票期間退款且金額大於 0**：開立折讓單並列印

### 注意事項
- 若服務銷售單的服務料件細項價值尚未攤算，會提示聯繫總部人員處理。
- 若銷售使用的包月服務已退貨，無法再對此銷售進行退貨。
- 退款期間無法關閉對話框（直到完成退款作業）。
- 若尚未新增任何退款紀錄就要離開，系統會自動取消銷退單。
- ONE 碼付款無法退超過 180 天的交易，需改用暫退款並退現金。

## 2. 包月退款

### 適用情境
退還已購買的包月/套餐服務。可自行填入退款金額，系統會平均分攤到包月服務內含項目中。

### 操作步驟

#### 步驟 1：點擊「退包月服務」按鈕
在結帳分頁中，點擊「退包月服務」按鈕（紫色）。

#### 步驟 2：搜尋會員
輸入以下資訊搜尋：
- 會員編號
- 會員姓名
- 手機號碼
- 康記額外聯絡資訊

#### 步驟 3：選擇包月服務
系統列出該會員在本門市的已購買包月服務（含已付款、部分退款、全部退款狀態的購買包月服務銷售）。選擇要退款的包月服務。

#### 步驟 4：確認退貨內容
系統顯示該包月服務的退貨項目與金額，確認後進入退款作業。

#### 步驟 5：退款操作
進入退款畫面，包月退款僅支援以下退款方式：
- **現金退款**
- **暫退款**（暫時不退還現金，記錄在案）

逐筆設定退款金額後完成退款。

#### 步驟 6：列印折讓單
退款完成後，系統自動列印折讓單（銷售編號含 KG 開頭的除外）。

### 注意事項
- 包月退款僅能退現金或使用暫退款，無法退回信用卡。
- 退款期間無法關閉對話框（直到完成退款作業）。
- 若尚未新增任何退款紀錄就要離開，系統會自動取消銷退單。
- 退款金額會平均分攤至包月服務包含的各個項目中。
