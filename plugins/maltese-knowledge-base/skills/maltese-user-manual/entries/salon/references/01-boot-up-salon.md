# 開機與登入（美容商品結帳）

## 適用情境
操作員需要開啟美容商品結帳介面，準備進行美容服務銷售。

## 操作步驟

### 步驟 1：開機與登入
美容商品結帳的開機與登入流程與一般商品結帳相同，請參閱一般商品入口的開機與登入說明。

### 步驟 2：選擇結帳入口
登入完成後，在主選單畫面中選擇「美容商品結帳」（而非「一般商品結帳」）。

### 步驟 3：進入美容服務 POS 平台
系統載入完成後，畫面標題顯示「萬達寵物 美容服務 POS 平台」，即表示已成功進入美容商品結帳介面。

## 注意事項
- 若系統尚在載入中（顯示讀取畫面），請耐心等候，切勿重複點擊。
- 若載入後自動跳回主選單，表示帳號驗證未完成或機台設定有誤，請聯繫主管或系統管理員。
- 美容入口與一般商品入口的介面佈局不同：美容入口僅有「結帳」與「管理」兩個分頁，沒有「商品」分頁。
