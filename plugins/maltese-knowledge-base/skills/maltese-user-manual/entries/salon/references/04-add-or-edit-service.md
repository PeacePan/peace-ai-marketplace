# 新增/修改服務

## 適用情境
操作員需要為會員的寵物新增美容服務項目，或修改已新增的服務內容。這是美容入口最核心的操作流程。

## 進入方式
在結帳分頁中，點擊「新增/修改服務」按鈕（藍色大按鈕，佔據兩格寬度）。

---

## 場景 1：選擇寵物

### 步驟 1：查看寵物列表
進入後，系統會顯示該會員名下所有「飼養中」的寵物列表，包含以下資訊：
- 寵物名稱
- 種類（貓/狗等）
- 品種
- 寵物性別
- 年齡
- 標籤

### 步驟 2：選擇寵物
點擊選取要接受服務的寵物，然後點擊「確定」。

### 注意事項
- 已經在服務清單中的寵物會顯示為禁用狀態，無法重複選取。
- 若需要新增寵物，點擊下方的「新增寵物」按鈕。

### 新增寵物
點擊「新增寵物」後，進入新增寵物表單：
1. 填寫寵物基本資料
2. 點擊「確定新增」完成新增
3. 系統自動返回寵物選擇畫面，可選取剛新增的寵物

---

## 場景 2：服務總覽

選擇寵物後，進入服務總覽畫面。此畫面分為兩個區域：

### 左側：服務項目列表
- 以分頁方式顯示每隻已選擇的寵物
- 每隻寵物會顯示其基本資料與已新增的服務
- 上方有搜尋列，可直接輸入服務編號搜尋並新增服務
- 美容服務與使用包月服務會顯示「排定日期」

### 右側：服務分類按鈕
提供以下分類選項，點擊後可瀏覽對應的服務目錄：

| 按鈕 | 說明 |
|------|------|
| 帶入消費 | 帶入該寵物的歷史消費紀錄 |
| 包月服務 | 瀏覽並購買包月/套餐服務 |
| 洗澡 | 單次洗澡服務 |
| 美容 | 單次美容服務 |
| 接送 | 接送服務 |
| 單次 | 其他單次服務 |
| 押金 | 押金服務 |
| 加減項 | 加項或減項服務 |
| 住宿 | 住宿類服務（以天計價） |
| 安親 | 安親類服務（以小時計價） |

### 底部按鈕列
- **使用包月服務**：使用已購買的包月服務次數
- **其他**：其他類型的服務
- **調整排定日期**：修改該寵物服務的排定日期
- **清除此寵物服務項目**：移除該寵物的所有已新增服務
- **確定**：確認所有服務並返回主畫面
- 可透過分頁上的「+」按鈕新增另一隻寵物

---

## 場景 3a：選擇服務項目

點擊分類按鈕後，系統顯示該分類下的服務列表：

### 步驟 1：瀏覽服務
表格顯示以下資訊：
- 項次
- 服務編號
- 服務名稱
- 工序類型（以標籤顯示）
- 售價範圍
- 每日上限金額（僅安親類服務顯示）

### 步驟 2：選擇服務
- 若搜尋結果有多項：點擊選取想要的服務項目，再點擊「確定」。
- 若搜尋結果只有一項：系統自動進入下一場景（設定價格畫面），無需手動選取。

---

## 場景 3b：設定價格與數量（修改服務）

選擇服務後，進入價格設定畫面：

### 步驟 1：檢視服務資訊
畫面顯示：
- 服務編號
- 服務名稱
- 價格區間（系統預設為區間中間值）

### 步驟 2：輸入售價
使用右側數字鍵盤輸入售價金額。售價必須在價格區間內。

### 步驟 3：填寫備註（選填）
可輸入收費原因或其他備註。若為「加項」或「減項」服務，備註為必填欄位。

### 步驟 4：確定
點擊「確定」將服務加入清單。

### 住宿/安親類服務的額外設定
- **住宿**：需額外選擇住宿天數（1~30 天）與起始/結束時間。住宿以天計價，金額 = 單價 x 天數。
- **安親**：需額外選擇安親時數（1~24 小時，可含半小時）與起始/結束時間。安親以小時計價，金額 = 單價 x 時數，但不超過每日上限金額。

---

## 場景 4：購買包月服務

### 步驟 1：瀏覽包月服務列表
點擊「包月服務」按鈕後，系統依寵物的種類與品種列出可購買的包月服務，顯示：
- 包月服務編號
- 包月服務名稱
- 使用次數
- 售價範圍

### 步驟 2：選擇包月服務
選取要購買的包月服務。可點擊「查看包月服務」按鈕查看該包月服務包含的細項。

### 步驟 3：設定價格與數量
選擇後進入修改服務畫面，可設定：
- 售價
- 購買數量（上限 15 組）

### 步驟 4：確定
確認後加入服務清單。

### 注意事項
- 員工（員購）無法購買包月服務。
- 同一隻寵物在同一門市的有效包月服務上限為 15 組（含購物車中的數量）。
- 購買包月服務時，購物車中不得有非包月服務與加減項以外的服務。
- 購買包月服務與使用包月服務不可同時存在於購物車中。

---

## 場景 5：使用包月服務

### 步驟 1：查看已購買的包月服務
點擊「使用包月服務」按鈕後，系統列出該會員於此門市、此寵物的已購買包月服務：
- 銷售編號
- 包月服務流水號
- 包月服務名稱
- 狀態（已付款/未付款）
- 剩餘次數
- 包月綁定的門市

### 步驟 2：選擇要使用的包月服務
選取後點擊「下一步」。

### 步驟 3：選擇使用的服務項目
系統顯示該包月服務包含的服務項目，選取後點擊「確定」，服務自動加入清單。

### 注意事項
- 剩餘次數為 0 的包月服務無法使用。
- 已退款的包月服務無法使用。
- 使用未付款的包月服務時，僅能使用一筆，且購物車總金額必須為 0 元。
- 購物車中已有購買包月服務時，無法同時使用包月服務。

---

## 場景 6：帶入前次消費

### 步驟 1：查看歷史消費紀錄
點擊「帶入消費」按鈕後，系統列出該寵物過往的消費紀錄，以年份分頁顯示。每筆紀錄包含：
- 發票號碼
- 銷售狀態
- 門市
- 銷售建立時間
- 銷售付款時間
- 服務項目明細
- 折扣明細

### 步驟 2：選擇要帶入的服務
在歷史紀錄中，點擊服務項目旁的加入按鈕，即可帶入該服務。

### 步驟 3：設定價格
帶入後會進入修改服務畫面，系統自動帶入上次的售價，可依需要調整。

### 注意事項
- 僅顯示「非未付款」的歷史消費紀錄。
- 若服務料件的活動期間已結束，無法再次帶入。
- 帶入的包月服務也需遵守包月服務的購買限制。

---

## 服務項目在中間欄的顯示

確認服務後返回主畫面，中間欄依寵物分組顯示所有已新增的服務：
- 每隻寵物有獨立的區塊，顯示寵物名稱
- 服務類型為「美容服務」或「使用包月服務」時，標題後會顯示「排定日期」
- 列表會自動捲動至最新新增的項目

## 編輯已新增的服務
再次點擊「新增/修改服務」按鈕，或點擊中間欄的服務項目，即可進入服務總覽畫面進行編輯。

## 移除服務
在服務總覽畫面中，每個服務項目右側有「刪除」按鈕，可移除單一服務。

### 注意事項
- 已使用過的包月服務（在未付款銷售中）無法移除。
- 一隻寵物只能有一組單次洗澡與一組單次美容項目，不可重複。
- 單獨購買加減項是不被允許的，必須搭配其他單次項目或包月服務。
- 每隻寵物的服務總金額不得為負數。
