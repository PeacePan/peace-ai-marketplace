# 查詢與搜尋功能

## 適用情境
操作員需要查詢歷史銷售紀錄、今日工單或取回未付款銷售進行付款。

---

## 1. 銷售查詢

### 適用情境
查詢今日工作日內的美容服務銷售紀錄。

### 操作步驟

#### 步驟 1：點擊「銷售查詢」按鈕
在結帳分頁中，點擊「銷售查詢」按鈕（紅色）。

#### 步驟 2：瀏覽銷售列表
系統自動載入今日工作日於本門市建立的服務銷售紀錄。

#### 步驟 3：查看銷售詳情
選取任一筆銷售紀錄，可查看完整銷售明細，包含服務項目、付款方式等資訊。

---

## 2. 工單列表

### 適用情境
查看今日有效的美容工單，通常由工作臺操作員使用，以確認服務進度或取回銷售。

### 操作步驟

#### 步驟 1：點擊「工單列表」按鈕
在結帳分頁中，點擊「工單列表」按鈕（紫色）。

#### 步驟 2：瀏覽工單
系統顯示今日有效的工單列表（僅顯示排定日期在今日工作日內的工單），包含以下資訊：

| 欄位 | 說明 |
|------|------|
| 工單編號 | 工單的唯一編號 |
| 工單建立時間 | 工單建立的日期與時間 |
| 銷售編號 | 對應的服務銷售編號 |
| 會員名稱 | 該筆銷售的會員 |
| 銷售狀態 | 未付款/已付款等 |
| 寵物名稱 | 涉及的寵物 |

#### 步驟 3：查看銷售詳情
選取工單後，點擊「查看銷售」按鈕，可查看該工單對應的服務銷售明細。

---

## 3. 查詢未付款銷售

### 適用情境
查詢本門市所有狀態為「未付款」的服務銷售，以便取回進行付款或取消。

### 操作步驟

#### 步驟 1：點擊「查詢未付款銷售」按鈕
在結帳分頁中，點擊「查詢未付款銷售」按鈕（靛藍色）。

#### 步驟 2：瀏覽未付款銷售清單
系統顯示所有未付款銷售，包含以下資訊：

| 欄位 | 說明 |
|------|------|
| 銷售編號 | 銷售的唯一編號 |
| 銷售建立時間 | 建立日期與時間 |
| 會員名稱 | 會員或員工姓名 |
| 寵物名稱 | 涉及的寵物 |
| 銷售類型 | 銷售的分類 |
| 銷售金額 | 應付總金額 |
| 銷售狀態 | 顯示為「未付款」 |
| 門市/機台 | 建立此銷售的門市與機台 |

#### 步驟 3：操作選項
選取銷售後，可執行以下操作：

| 按鈕 | 說明 |
|------|------|
| 加入銷售 | 將此筆未付款銷售載入畫面，可繼續編輯或點擊「小計」進行付款 |
| 查看銷售 | 查看該筆銷售的完整詳情 |
| 取消銷售 | 取消此筆未付款銷售（需二次確認） |
| 帶入會員 | 僅帶入該銷售的會員資料（不載入銷售內容） |

### 加入銷售的流程
1. 選取目標銷售，點擊「加入銷售」
2. 系統重新取得該銷售的最新資料（避免其他機台已修改）
3. 自動載入會員、寵物、服務項目至畫面
4. 可編輯服務後點擊「小計」完成付款
5. 對話框自動關閉

### 取消銷售的流程
1. 選取目標銷售，點擊「取消銷售」
2. 系統跳出二次確認對話框：「請再次確認是否要取消未付款銷售」
3. 確認後銷售被取消，清單自動更新
4. 若該銷售有使用包月服務且關聯的銷售尚未取消，需先取消關聯銷售

---

## 注意事項
- 銷售查詢範圍限於本門市今日工作日內的資料。
- 工單列表會過濾排定日期不在今日工作日內的工單。
- 查詢未付款銷售載入時，系統會重新取得最新資料，避免資料不一致。
- 在其他機台進行中的銷售，取回後可能需要注意資料版本衝突。
