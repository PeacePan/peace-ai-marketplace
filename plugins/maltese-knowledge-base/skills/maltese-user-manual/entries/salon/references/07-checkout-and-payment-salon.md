# 結帳與付款

## 適用情境
操作員已新增服務項目，準備進行付款結帳。

## 操作步驟

### 步驟 1：確認服務項目
在結帳分頁中，確認中間欄的服務項目列表與底部的小計金額無誤。

### 步驟 2：點擊「小計」按鈕
點擊底部右側的「小計」按鈕（粉紅色，佔 65% 寬度）。

### 步驟 3：系統驗證
系統會進行以下驗證：
1. 是否已開帳
2. 是否已新增服務項目
3. 工作臺模式是否僅為 0 元銷售
4. 查價機模式無法使用此功能
5. 是否跨日未清帳
6. 是否有使用包月服務而原購買銷售尚未付款
7. 發票機連線是否正常

### 步驟 4：進入付款畫面
驗證通過後，系統建立或更新銷售單，畫面右側滑出付款側邊欄。

---

## 付款側邊欄

### 左側：服務項目確認區
顯示本次銷售的完整服務項目，依寵物分組，包含：
- 服務項目明細
- 排定日期
- 整單折扣列表（不可刪除）
- 點數禮物卡列表（不可刪除）
- 已選擇的付款方式與金額列表

### 右側：付款方式按鈕區

#### 現金付款
| 按鈕 | 說明 |
|------|------|
| 現金 | 手動輸入現金金額 |
| 現金 500 | 快速輸入 500 元 |
| 現金 1000 | 快速輸入 1,000 元 |
| 現金 1500 | 快速輸入 1,500 元 |
| 現金 2000 | 快速輸入 2,000 元 |

#### 信用卡付款
| 按鈕 | 說明 |
|------|------|
| 離線信用卡（全額） | 以離線信用卡支付全部餘額 |
| 離線信用卡（指定金額） | 手動輸入離線信用卡金額 |
| 信用卡（全額） | 以信用卡支付全部餘額 |
| 信用卡（指定金額） | 手動輸入信用卡金額 |

#### 其他付款方式
| 按鈕 | 說明 |
|------|------|
| One 碼 | 台新 One 碼支付（全額） |
| 宜睿票券 | 使用宜睿禮券支付 |

#### 其他功能
| 按鈕 | 說明 |
|------|------|
| 發票明細列印 | 切換列印/關閉（僅限門市類型，非專櫃） |
| 輸入發票方式 | 設定統編或載具等發票資訊 |
| 清除發票方式 | 清除已設定的發票資訊 |

---

## 付款方式限制

### 信用卡
- 同一筆銷售只能使用一張信用卡。若已使用信用卡支付部分款項，不可再搭配其他信用卡。

### One 碼
- One 碼必須全額支付，不得搭配其他付款方式。
- 若已有其他付款記錄（宜睿票券除外），無法使用 One 碼。
- 若使用宜睿票券，餘額僅可透過現金/信用卡結帳，不可使用 One 碼。

### 宜睿票券
- 不得搭配 One 碼使用。
- 專櫃門市無法使用。
- 購買包月服務、安親、住宿時不適用。
- 需要門市已設定宜睿禮券的用戶編號與密鑰。

### 專櫃門市
- 專櫃門市無法使用信用卡、離線信用卡、One 碼、宜睿票券，需至專櫃 POS 機進行。

---

## 刪除付款記錄
在付款記錄列表中，部分付款方式右側有刪除按鈕，可移除該筆付款：
- 現金、離線信用卡等可刪除
- 信用卡（線上）、One 碼、宜睿票券等第三方支付無法刪除

---

## 完成結帳

### 步驟 1：確認付款金額
當付款金額達到或超過應付總額時，找零金額會自動計算並顯示在列表中。

### 步驟 2：點擊結帳
點擊結帳按鈕完成交易。若付款方式包含現金，系統會自動開啟錢櫃。

### 步驟 3：收據/發票列印
結帳完成後，系統會依設定自動列印收據或發票。

---

## 發票明細列印規則
- 銷售總金額為 0 元時：一律列印明細
- 有統編時：預設列印明細
- 無統編且金額大於 0 時：預設不列印明細
- 操作員可手動切換列印/關閉

---

## 返回
在付款畫面中，可點擊返回按鈕回到結帳分頁。系統行為：
- 若此銷售是從未付款銷售取回的：返回時不會取消銷售單
- 若此銷售是新建立的：返回時會自動取消銷售單

---

## 與一般商品結帳的差異
- 美容入口銷售的是服務而非實體商品
- 付款方式類型針對美容服務設計
- 服務項目依寵物分組顯示
- 有工作臺模式的限制（僅 0 元可結帳）
- 有跨日未清帳的檢查機制
