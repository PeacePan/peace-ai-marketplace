# 開帳

## 適用情境
每日營業前，操作員需要進行開帳作業，系統才允許新增美容服務與結帳。

## 操作步驟

### 步驟 1：切換到管理分頁
點擊畫面右側的「管理」分頁。

### 步驟 2：點擊「開帳」按鈕
在管理面板中，點擊「開帳」按鈕。

### 步驟 3：確認開帳資訊
系統會顯示開帳對話框，同時載入近 5 日的開帳紀錄供參考。確認資訊無誤後完成開帳。

## 未開帳時的行為
- 若尚未開帳，中間欄的服務項目列表上方會出現遮罩，顯示「請先至管理面板進行開帳」。
- 此時無法新增任何服務項目，必須先完成開帳。
- 注意：若機台模式為「查價機」，則不受開帳限制。

## 修改開帳紀錄

### 步驟 1：點擊「開帳紀錄修改」
在管理分頁中，點擊「開帳紀錄修改」按鈕。

### 步驟 2：查看並修改
系統會載入近 5 日的開帳紀錄與目前機台設定，供您查看與修改。

## 注意事項
- 若系統偵測到已開帳但尚未清帳，點擊「開帳」時會提示「請先進行清帳」，並自動開啟清帳對話框。
- 開帳只需每日執行一次，重複開帳前須先清帳。
- 閒置逾時（美容入口為 30 分鐘）時系統會自動登出操作員，但不影響開帳狀態。
