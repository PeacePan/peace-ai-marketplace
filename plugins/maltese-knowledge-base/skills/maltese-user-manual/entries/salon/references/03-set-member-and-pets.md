# 設定會員與寵物

## 適用情境
美容入口必須先設定會員後，才能新增服務項目。這是與一般商品入口的重要差異——一般商品入口可在未設定會員的情況下直接掃碼加入商品，但美容入口不行。

## 操作步驟

### 步驟 1：開啟會員查詢
畫面左上方為會員資訊區域。點擊該區域開啟會員查詢。

### 步驟 2：搜尋會員
您可透過以下方式搜尋會員：
- 輸入會員電話號碼
- 輸入會員姓名
- 輸入會員編號

### 步驟 3：選擇會員
從搜尋結果中選擇正確的會員。選擇後，系統會自動載入該會員的相關資訊。

### 步驟 4：確認會員資訊
設定會員後，左上方會顯示：
- 會員名稱
- 會員等級
- 會員點數
- 該會員擁有的寵物列表

## 清除會員
若需更換會員，可點擊會員資訊區域右側的清除圖示，移除目前設定的會員。

## 注意事項
- 美容入口**必須先設定會員**，才能執行「新增/修改服務」。未設定會員就點擊新增服務時，系統無法進入寵物選擇畫面。
- 支援員工購買（員購），此時設定的是員工而非一般會員。員購身份有部分限制，例如無法購買包月服務、無法使用折扣碼。
- 設定會員後，系統會同步載入該會員所屬的寵物資料，供後續選擇服務時使用。
- 若會員尚未註冊任何寵物，可在新增服務流程中直接新增寵物（詳見「新增/修改服務」說明）。
