# 美容工作臺 Agent 指引

## 入口資訊
- 網址：`/workbench`
- 頁面標題：「萬達寵物 美容工作臺」

## 介面佈局

美容工作臺採上下兩區佈局：

| 區域 | 位置 | 內容 |
|------|------|------|
| 頂部統計列 | 最上方 | 左側為工單統計（總單數、貓數、狗數）；右側為門市名稱、機台名稱、即時時間、離開按鈕 |
| 工具列 | 統計列下方 | 左側顯示目前選取的美容師姓名；右側為四個按鈕：重新整理、新增/變更美容師、查詢其他日期、完成 |
| 工單卡片區 | 主要區域 | 可捲動的工單卡片清單，每張卡片顯示工單資訊與服務任務按鈕 |

### 工單卡片結構

每張工單卡片包含三個區塊：
1. **基本資訊列**：工單編號（含包月/單次標示）、工單排定時間、銷售狀態、銷售編號
2. **寵物與飼主資訊列**：寵物名稱、飼主電話、寵物種類、寵物品種、飼主名稱
3. **服務任務區**：以 10 欄網格排列的工序按鈕，每個按鈕顯示工序名稱與已指派美容師姓名

## 與其他入口的差異

- **非銷售介面**：工作臺不進行結帳、付款、退款等銷售操作，僅進行美容師指派。
- **無商品或服務新增**：無法新增或修改服務項目，僅能查看已建立的工單。
- **無會員設定**：不需要設定會員，直接顯示所有工單。
- **無開帳/清帳**：不涉及帳務管理功能。
- **無分頁**：僅有單一主畫面，無結帳/管理等分頁切換。
- **資料來源**：工單來自美容商品結帳入口的後付款銷售，付款也在美容商品結帳入口完成。
- **日期切換**：可查詢任意日期的工單（美容商品結帳入口僅能操作當日銷售）。

## 重要說明

- 工作臺的核心功能是「為美容服務工單指派美容師」，所有操作都圍繞此目的。
- 指派操作採「先選美容師，再點工序按鈕」的模式，點擊同一美容師的已指派按鈕等同取消指派。
- 所有指派修改皆為暫存，必須按「完成」按鈕才會寫入系統。
- 系統僅傳送實際有差異的指派變更，若修改後又改回原值則不計入變更。
- 重新整理與查詢其他日期前，若有未儲存的變更，系統會先確認是否捨棄。
- 非本日工單若已有美容師指派則不可更改，僅可查看。
- 無效工單（已退款/已作廢）會以灰色遮罩顯示，無法進行任何指派操作。
- 統計列數字包含所有狀態的工單（含無效工單）。
