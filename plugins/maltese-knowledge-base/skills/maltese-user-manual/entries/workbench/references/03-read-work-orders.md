# 閱讀工單卡片

## 適用情境
操作員需要了解每張工單卡片上顯示的資訊，以及不同狀態下的外觀差異。

## 操作步驟

### 步驟 1：檢視工單基本資訊
每張工單卡片的第一列顯示以下資訊：

- **工單編號**：系統自動編排的序號，後方括號內標示「包月」或「單次」，表示此工單是使用包月服務或單次服務。
- **工單排定時間**：此工單預定執行服務的日期。
- **銷售狀態**：顯示該筆銷售的目前狀態（如「未付款」或「已付款」等）。
- **銷售編號**：對應的銷售單編號，可用於在美容商品結帳入口查詢該筆銷售。

### 步驟 2：檢視寵物與飼主資訊
第二列顯示以下資訊：

- **寵物名稱**：接受服務的寵物名稱。
- **飼主電話**：飼主的聯絡手機號碼。
- **寵物種類**：貓或狗。
- **寵物品種**：具體品種名稱。
- **飼主名稱**：飼主姓名。

### 步驟 3：檢視服務任務區
卡片下方為服務任務按鈕區，以網格排列。每個按鈕代表一項服務工序（如洗澡、剪毛等），按鈕上方顯示工序名稱，下方以大字顯示已指派的美容師姓名。若該工序尚未指派美容師，則下方為空白。

## 注意事項
- **未付款工單**：銷售狀態文字會以醒目的警示顏色顯示，提醒操作員該筆銷售尚未付款。
- **已付款工單**：銷售狀態文字以一般的強調色顯示。
- **無效工單**（已退款或已作廢）：整張工單卡片會被半透明灰色遮罩覆蓋，工序按鈕無法點擊，表示該工單已失效，不需指派美容師。
- 當美容師指派與原始記錄不同時（已變更但尚未儲存），美容師姓名會以強調色標示，方便識別哪些指派已被修改。
