# 開機與登入（美容工作臺）

## 適用情境
操作員需要開啟美容工作臺介面，準備為今日的美容服務工單指派美容師。

## 操作步驟

### 步驟 1：開機與登入
美容工作臺的開機與登入流程與一般商品結帳相同，請參閱一般商品入口的開機與登入說明。

### 步驟 2：選擇入口
登入完成後，在主選單畫面中選擇「美容工作臺」（而非「一般商品結帳」或「美容商品結帳」）。

### 步驟 3：進入美容工作臺
系統載入完成後，畫面標題顯示「萬達寵物 美容工作臺」，即表示已成功進入工作臺介面。系統會自動載入今日的美容服務工單。

## 注意事項
- 若系統尚在載入中（顯示讀取動畫），請耐心等候，切勿重複點擊。
- 若載入後自動跳回主選單，表示帳號驗證未完成或機台設定有誤，請聯繫主管或系統管理員。
- 美容工作臺不是銷售介面，無法進行結帳或付款操作。其唯一功能是為美容服務工單指派美容師。
