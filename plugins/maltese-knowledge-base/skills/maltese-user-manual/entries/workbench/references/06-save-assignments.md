# 儲存指派結果

## 適用情境
操作員完成美容師指派後，需要將變更儲存至系統。

## 操作步驟

### 步驟 1：點擊「完成」按鈕
確認所有指派無誤後，點擊工具列右側的「完成」按鈕。

### 步驟 2：系統自動比對差異
系統會自動比對目前畫面上的指派與原始資料的差異，只將有實際變更的部分傳送至系統儲存。未修改的任務不會被重複傳送。

### 步驟 3：確認儲存成功
儲存成功後，系統會清除所有暫存的變更狀態。畫面上的美容師指派已正式寫入系統。

## 注意事項
- **無變更時**：若您在畫面上沒有做任何指派修改就按「完成」，系統會提示「沒有變更任何工序的美容師」，不會進行任何傳送。
- **僅傳送差異**：即使您修改了多張工單，系統只會傳送實際與原始記錄不同的部分。若您先修改再改回原值，該筆不會被視為變更。
- **儲存期間按鈕停用**：儲存過程中，工具列上的所有按鈕會暫時停用，請等待儲存完成。
- **儲存失敗**：若網路異常或系統錯誤導致儲存失敗，暫存的變更仍保留在畫面上，您可以排除問題後再次按「完成」重試。
