# 重新整理與查詢其他日期

## 適用情境
操作員需要重新載入最新的工單資料，或查看其他日期的工單。

## 操作步驟

### 步驟 1：重新整理目前工單
點擊工具列上的「重新整理」按鈕，系統會重新從伺服器載入目前日期的工單資料。

- **有未儲存變更時**：系統會彈出確認視窗，提示「重新整理後會捨棄目前所有變更，確定要執行此動作？」。點擊「確定」會捨棄所有未儲存的指派變更並重新載入；點擊「取消」則回到原畫面，變更保留。
- **無未儲存變更時**：直接重新載入，不會出現確認視窗。

### 步驟 2：查詢其他日期的工單
點擊工具列上的「查詢其他日期」按鈕：

1. **有未儲存變更時**：同樣會先彈出確認視窗，提示捨棄變更。確認後才進入日期選擇。
2. 系統彈出日期選擇視窗，顯示「帳務日」日期選擇器，預設為今日日期。
3. 選擇目標日期後，點擊「確定」，系統載入該日期的工單。點擊「關閉」則取消操作。

## 注意事項
- 重新整理與查詢其他日期都會觸發未儲存變更檢查，確保您不會意外遺失已做的指派。
- 查詢其他日期後，頂部統計列的數字會更新為該日期的工單統計。
- 非本日的工單若已有美容師指派，將無法更改（僅可查看）。
- 載入過程中畫面會顯示讀取動畫，所有按鈕暫時停用。
