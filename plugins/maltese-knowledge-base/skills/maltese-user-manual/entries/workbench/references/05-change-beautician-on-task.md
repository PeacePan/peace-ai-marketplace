# 變更或移除任務上的美容師

## 適用情境
操作員需要更換某個服務任務已指派的美容師，或移除該任務的美容師指派。

## 操作步驟

### 步驟 1：變更美容師（替換為另一位）
1. 點擊工具列的「新增 / 變更美容師」，選擇新的美容師。
2. 點擊目標工序按鈕，原本的美容師姓名會被新選取的美容師覆蓋。

### 步驟 2：移除美容師指派
若要將某個工序的美容師移除（回到未指派狀態）：

1. 確認工具列上目前選取的美容師與該工序上顯示的美容師為**同一人**。
2. 點擊該工序按鈕，系統會自動將指派清除，按鈕下方的美容師姓名消失。

此行為的邏輯為：當目前選取的美容師與工序上的美容師相同時，點擊等同於「取消指派」。

## 注意事項
- 變更與移除操作同樣只是暫存在畫面上，需按「完成」才會寫入系統。
- 非本日的工單若已有美容師指派，系統不允許更改，會提示「不可更改非本日工單的美容師」。
- 已變更但尚未儲存的指派會以強調色標示，方便辨識哪些任務已被修改。
