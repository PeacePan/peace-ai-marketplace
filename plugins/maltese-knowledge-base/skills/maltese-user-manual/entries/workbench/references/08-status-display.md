# 工單狀態顯示

## 適用情境
操作員需要了解工單卡片上不同狀態的視覺差異，以及頂部統計數字的含義。

## 操作步驟

### 步驟 1：辨識未付款工單
銷售狀態欄位顯示「未付款」時，文字以警示色（紅色系）顯示，與其他狀態明顯不同。這表示此筆銷售尚未在美容商品結帳入口完成付款。

### 步驟 2：辨識已付款工單
已付款的工單，銷售狀態文字以一般的強調色（藍色系）顯示，外觀與其他欄位一致。

### 步驟 3：辨識無效工單
已退款或已作廢的工單會被標記為無效。無效工單整張卡片覆蓋半透明灰色遮罩，所有工序按鈕無法點擊。此類工單不需指派美容師。

### 步驟 4：判讀頂部統計列
頂部統計列顯示三個數字：

- **總單數**：目前載入日期的所有工單總數（包含未付款、已付款及無效工單）。
- **貓**：工單中寵物種類為貓的工單數量。
- **狗**：工單中寵物種類為狗的工單數量。

## 注意事項
- 統計列的數字包含所有狀態的工單（含無效工單），並非只計算有效工單。
- 付款操作不在工作臺進行。若您發現工單為未付款狀態，需在美容商品結帳入口查詢該銷售編號並完成付款。
- 無效工單雖然無法操作，但仍會顯示在工單列表中並計入統計數字，方便查閱記錄。
- 包月服務與單次服務的工單在工單編號旁會以括號標示「包月」或「單次」，方便區分。
