# 指派美容師至服務任務

## 適用情境
操作員需要為工單中的服務任務指派負責的美容師。

## 操作步驟

### 步驟 1：選擇美容師
點擊工具列上的「新增 / 變更美容師」按鈕，系統會彈出美容師選擇視窗。視窗中顯示美容師清單表格，包含「員工編號」與「姓名」兩欄。

### 步驟 2：從清單選取或新增美容師
- **從清單選取**：點擊表格中的美容師列將其選取（選取後該列會反白），然後點擊「確定」。
- **新增美容師**：若清單中沒有目標美容師，點擊「新增」按鈕，畫面切換至數字鍵盤，輸入該美容師的員工編號後點擊確認。系統會查詢並加入該美容師。

選取完成後，工具列左側的「美容師」欄位會顯示已選取的美容師姓名。

### 步驟 3：點擊服務任務按鈕完成指派
在工單卡片的服務任務區域，直接點擊要指派的工序按鈕。點擊後，該按鈕下方會立即顯示目前選取的美容師姓名，表示指派成功。

### 步驟 4：為多個任務指派同一位美容師
選取美容師後，可連續點擊不同工單上的不同工序按鈕，快速將同一位美容師指派給多個任務。

### 步驟 5：為不同任務指派不同美容師
若需要讓不同的工序由不同美容師負責，先回到「新增 / 變更美容師」選擇另一位美容師，再點擊對應的工序按鈕即可。

## 注意事項
- 指派前必須先選擇美容師。若尚未選擇就直接點擊工序按鈕，系統會提示「請先設定美容師」。
- 若清單中新增的員工編號已存在，系統會提示「已存在該使用者」。
- 所有指派操作僅在畫面上暫存，尚未寫入系統。務必記得最後按「完成」按鈕儲存（請參閱「儲存指派結果」說明）。
- 無效工單（有灰色遮罩）的工序按鈕無法點擊。
