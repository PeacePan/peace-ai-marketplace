# 美容工作臺介面總覽

## 適用情境
操作員進入美容工作臺後，需要了解畫面各區域的功能與用途。

## 操作步驟

### 步驟 1：認識頂部統計列
畫面最上方的統計列分為左右兩區：

- **左側統計區**：顯示「總單數」、「貓」數量、「狗」數量。這三個數字分別代表目前載入日期的全部工單總數、貓的工單數、狗的工單數。
- **右側資訊區**：顯示「門市」名稱、「機台」名稱、「時間」（即時更新的目前日期時間），以及一個離開按鈕（點擊後回到主選單）。

### 步驟 2：認識工具列
統計列下方為工具列，包含以下元素：

- **目前美容師**：左側顯示「美容師」及目前已選取的美容師姓名；若尚未選取則顯示「無」。
- **重新整理**：重新從系統載入工單資料。
- **新增 / 變更美容師**：開啟美容師選擇視窗，選定要指派的美容師。
- **查詢其他日期**：開啟日期選擇器，載入其他日期的工單。
- **完成**：將目前所有美容師指派變更儲存至系統。

### 步驟 3：認識工單卡片區
工具列下方為工單卡片網格區域，以可捲動的清單方式顯示每一張工單。每張工單卡片的詳細內容請參閱「閱讀工單卡片」說明。載入中時畫面會顯示讀取動畫。

## 注意事項
- 工具列的四個按鈕在系統正在載入工單或正在儲存指派時會暫時停用（灰色不可點擊），請等候操作完成。
- 工單卡片區域的奇偶行有不同底色，方便區分相鄰工單。
- 離開按鈕會直接回到主選單，若有未儲存的指派變更將會遺失，請先按「完成」儲存。
